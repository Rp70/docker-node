# AGENTS

This repository uses automated assistance for maintenance and routine updates.

- **GitHub Copilot** — automated coding assistant used to: update Dockerfile templates, add new Node versions, run repository scripts, and create repository docs.

Notes:
- Changes made by the agent are small, explicit, and accompanied by tests or verification steps when required.
- For manual confirmation (for example, Docker Hub pushes), please verify credentials and CI results before merging.
